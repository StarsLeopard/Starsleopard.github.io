JS相关函数汇总

发表于 2023-07-07 Disqus：

JS笔记及函数汇总记忆

1.Browswer

Window Properties （window属性）

document (document对象)
history (history对象，包含了用户在浏览器窗口访问过的URL)
location (对象包含有关当前URL的信息)
navigator (对象包含浏览器有关信息，可不使用window前缀)
screen (对象包含客户端显示屏的信息)

name (用于设置和存放窗口名称)
parent (返回当前窗口的父窗口的window对象)
opener (返回对创建该窗口的window对象的引用，用于window.open()方法创建的窗口)
self (返回对窗口自身的只读引用)

innerHeight (窗口文档显示区域的高度)
innerWidth (窗口文档显示区域的宽度)
outerHeight (窗口外部高度，包括所有元素如滚动栏工具栏)
outerWidth (窗口内部高度，包括所有元素如滚动栏工具栏)
closed (监控窗口是否关闭)
pageXOffset (当前页相对窗口显示区左上角的X位置)
pageYOffset (当前页相对窗口显示区左上角的Y位置)
screenLeft (返回窗口相对于屏幕的X坐标)
screenTop (返回窗口相对于屏幕的Y坐标)（IE不支持）
screenX (返回窗口相对于屏幕的X坐标)
screenY (返回窗口相对于屏幕的Y坐标)（火狐不支持）

Window Methods （window方法）

alert() ：显示一个指定消息和确认按钮的警告框
confirm() ：显示一个带有指定消息和确认和取消按钮的对话框
prompt() ：显示可以提示用户进行输入的对话框

focus() ：设置焦点 （可以给DOM元素设置）
blur() ：把键盘焦点从顶层窗口移开 （可以清除DOM元素焦点）
close() ：关闭浏览器窗口
open() ：打开一个新的浏览器窗口或者查找一个已命名的窗口

print() ：打印当前浏览器窗口

setInterval() ：设置定时器
clearInterval() ：清除定时器
setTimeout() ：设置延时器
clearTimeout() ：清除延时器

moveBy() ：相对窗口当前坐标移动窗口
moveTo() ：把窗口移动到相对左上角的指定坐标
resizeBy() ：
resizeTo() ：窗口大小重新设置

scrollBy(x,y) ：可以把内容滚动到指定的像素数,x向右滚动像素，y向下滚动像素
scrollTo(x,y) ：可以把内动滚动到指定坐标

2.Basic Objects

Array Methods (数组方法)

concat(array2,...) ：链接两个或者多个数组，返回被连接数组的一个副本
indexOf(item,start) ：查找数组中首个出现的匹配项，或者字符串中首个出现的值
lastIndexOf(item,start) ：找最后一个匹配项
join(separator) ：把数组元素链接成一个字符串，参数为链接符号，默认逗号
pop() ：移除数组最后一个元素，返回移除的元素
shift() ：移除数组第一个元素，返回移除的元素
push() ：向末尾添加一个或多个元素，并返回新数组长度
unshift() ：想数组开头添加一个或更多元素，并返回新数组长度
reverse() ：颠倒数组元素顺序
slice(start,end) ：提取数组中某一段的元素，并返回提取出的新数组
toString() ：把数组转化为字符串，并返回结果，用逗号连接
map() ：创建一个新数组，这个新数组由原数组中的每个元素都调用一次提供的函数后的返回值组成

Math Properties And Methods（Math对象相关属性和方法）

Math.E ：自然对数底数
Math.PI ：圆周率

Math.abs() ：返回一个数的绝对值
Math.sin() ：返回一个数的正弦（单位为弧度）
Math.asin() ：返回一个值的反正弦

Math.sqrt() ：返回一个数的平方根
Math.cbrt() ：返回一个数的立方根
Math.ceil() ：返回大于给定值的最小整数
Math.floor() ：返回小于给定值的最大整数
Math.round() ：返回最接近的整数

Math.max() ：返回输入参数的最大值
Math.min() ：返回输入参数的最小值

Math.random() ：返回一个随机0-1中的浮点数（包括0,不包括1
Math.exp() ：返回 E 的 x 次幂的值, x为入参

String Methods （String对象方法）

charAt() ：返回指定位置的字符
charCodeAt() ：返回指定位置字符的Unicode编码
fromCharCode() ：将指定Unicode编码转化为对应字符
match(reg) ：在字符串中找到指定的值，可以用正则匹配
replace(str1,str2) ：字符串全局替换，将st1替换为str2
search(reg) ：检索字符串中指定的子字符串，或者与正则表达式相匹配的子字符串，返回第一个所在位置
split(separator) ：将字符串分割，separator为分割标志，默认每个字符都被分割
substr(start,length) ：在字符串中抽取，从下标开始的指定数目的字符
substring(from,to) ：提取两个指定下标之间的字符，包括开始字符，但不包括结束字符
toLowerCase() ：将字符串转换为小写
toUpperCase() ：将字符串转换为大写

Global Methods (全局方法)

encodeURL(url) ：对url编码进行编码，url中的特殊字符不会被编码，如/?
decodeURL(url) ：对url编码进行解码

encodeURLComponent() ：对字符串编码，全部编码为ASCLL码，除了 - _ . ! ~ * ' ( )
decodeURLComponent() ；对应解码

escape() ：字符串编码，除了ASCLL和* @ - _ + . /
unescape() ：解码

isNaN() ：是否不是数字
Number() ：将参数转化为数字，若不能则返回NaN
String() ：把对象转化为字符串

parseFloat() ：字符串首个为数字时，返回从左到右检索出的浮点数。若第首个不是数字则返回NaN
parseInt(string, radix) ：可解析一个字符串，并返回一个整数。"0x"开头时，解析为16进制
